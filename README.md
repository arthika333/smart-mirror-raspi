# Smart Mirror using Raspberry Pi

This project is a Smart Mirror system developed using the MagicMirror² framework
and Raspberry Pi. The system displays real-time information such as calendar
events, schedules, weather updates, and Spotify playback.

## Features Implemented
- Calendar and important dates display
- Weekly schedules and event tracking
- Weather and climate information
- Spotify integration (display module)
- Modular MagicMirror-based architecture

## Hardware Used
- Raspberry Pi 4
- Two-way mirror
- External display
- USB speaker (configured, playback in progress)

## Software Used
- MagicMirror²
- Node.js
- JavaScript, HTML, CSS
- Raspberry Pi OS

This project focuses on lifestyle assistance and information display.
